# Prompt-Based AI Demo

This is a simple AI demo using Groq LLM API.

- Enter a scientific term
- Get structured JSON output: Definition, Analogy, Real-life Examples
- UI built with Gradio

## How to run

1. Create a secret in Colab: GROQ_API_KEY
2. Install dependencies: !pip install gradio groq
3. Run app.py in Colab
